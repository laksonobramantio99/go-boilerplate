# go-boilerplate
`go-boilerplate` is a golang boilerplate for backend service.

## Features
- [x] use gin framework
- [x] have dev and prod config setup
- [ ] with separate api, consumer and cron binaries
- [ ] use kafka as the consumer
- [ ] use postgresql DB
- [ ] use redis
- [ ] hierarchy api - handler - usecase - repo
- [ ] use yaml config
